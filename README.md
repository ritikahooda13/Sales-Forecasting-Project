# 📊 Sales Forecasting Dashboard

## 🚀 Project Overview
This project is an end-to-end Sales Forecasting Dashboard built using Python, Streamlit, and Prophet. It helps visualize past sales data and predict future sales trends.

## 🛠️ Tech Stack
- Python
- Pandas
- Streamlit
- Prophet
- Matplotlib

## 📁 Files Included
- sales_dashboard.py → Streamlit dashboard
- sales_forecasting.py → forecasting model
- Sample - Superstore.csv → dataset

## ▶️ How to Run
```bash
streamlit run sales_dashboard.py
